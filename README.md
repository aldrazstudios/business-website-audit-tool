# Business Website Audit Tool

A simple website audit tool for restaurants, cafés, gyms, salons, clinics, and other local businesses.

## What it does

This tool checks if a local business website has the important features needed to look professional, build trust, and help visitors become customers.

## Features

- Website score out of 100
- Business type selection
- Contact and trust checklist
- Content quality checklist
- User experience checklist
- Automatic improvement recommendations
- Mobile-friendly design
- Built with HTML, CSS, and JavaScript

## Why I built it

Many local businesses have websites that look okay but miss important things like WhatsApp buttons, Google Maps links, opening hours, real photos, reviews, and clear service information.

This tool helps find those missing pieces quickly.
